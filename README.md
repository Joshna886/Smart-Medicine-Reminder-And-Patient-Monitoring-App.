
# Smart Medicine Alert System

## Overview
A Flutter-based mobile application designed to improve medication adherence through automated medicine reminders, appointment tracking, caretaker monitoring, and medicine stock management.

## Features
- Medicine Reminder Alerts
- Appointment & Medical Test Scheduling
- Caretaker Monitoring
- Medicine Stock Tracking
- Firebase Authentication
- Cloud Firestore Integration
- Real-Time Notifications

## Technologies Used
- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Cloud Messaging (FCM)
